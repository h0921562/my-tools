請求書作成

登録した会社の請求書を発行します。

請求書を作る
使い方
発行元の会社を選び、宛先と明細を入れて発行します。品目名から税率(8%/10%)が自動で入りますが、 必ず目で確認してください。PDFはGoogle Driveの「請求書データ」に、履歴は台帳シートに残ります。

インボイス登録番号が未設定の会社は、区分記載請求書の様式で発行されます(登録番号は印字されません)。

最初に接続先を設定します

URLの調べ方
スプレッドシート「請求書管理」→ 拡張機能 → Apps Script → 右上「デプロイ」→ 「デプロイを管理」→ ウェブアプリのURLをコピー。

このURLはこの端末のブラウザ内だけに保存され、GitHubには保存されません。 アプリ自体も「自分のみアクセス可」で公開しているので、URLを知られても他人は開けません。
